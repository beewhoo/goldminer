# GOLDMINER
- GOLDMINER CRM BACKEND API

## Local Project Setup
 * Pre-requisites:
    - PostgreSQL. Versions 9.3 and up are supported.
    - ruby version - 3.0.0
    - rails version - 6.1.3

 * Run bundle install
 * Run migration `rails db:setup`
 * Seed database `rails db:seed`
 * Run rails server using `rails s`
